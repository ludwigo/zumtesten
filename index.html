Hier mal ein Workflow:

    Verzeichnis erstellen
    git init
    Dateien/Projekt erstellen
    git add Dateien
    git commit -m "Kommentar"
    auf Github ein Remoteverzeichnis erstellen
    mit git remote add origin serveradresse-bei-github initialisieren
    mit git push -u origin master Sachen rüberschieben
    auf Cloudcontrol neue App erstellen
    mit cctrlapp Appname push rüberschieben
    mit cctrlapp Appname deploy veröffentlichen

und siehe da: es geht!
